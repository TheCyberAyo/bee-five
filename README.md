##🐝 BeeFive
BeeFive is a competitive Connect 5 multiplayer game built for web and mobile. It features real-time matchmaking, global and institutional leaderboards, and an ELO-based ranking system that rewards skill-based progression.

#🎮 Features
🧠 Connect 5 gameplay (strategy-based board game)
🏆 ELO ranking system – win or lose points based on performance
🌍 Global & institutional leaderboards
🟢 Online/offline player status tracking
🔎 Player search functionality
🔔 Opponent notifications (challenge invites & game updates)
📊 Competitive progression system
📈 Ranking System
Real-time matchmaking lobby system

BeeFive uses an ELO-based system where:
Winning increases your rating
Losing decreases your rating
You are placed on:
Global leaderboard 🌍
Institutional leaderboard 🏫

#📱 Platforms
Web version built with Next.js
Mobile version built with Flutter (Dart)
Both platforms share the same competitive ecosystem and player data.

#🎯 Project Purpose
BeeFive was built to create a competitive, skill-based multiplayer environment where players can:
Compete globally
Track progress over time
Engage in structured ranking systems
Challenge real opponents in real-time
