**🐝 BeeFive**</br>
BeeFive is a competitive Connect 5 multiplayer game built for web and mobile. It features real-time matchmaking, global and institutional leaderboards, and an ELO-based ranking system that rewards skill-based progression.

**Features** </br>
🧠 Connect 5 gameplay (strategy-based board game)</br>
🏆 ELO ranking system – win or lose points based on performance</br>
🌍 Global & institutional leaderboards</br>
🟢 Online/offline player status tracking</br>
🔎 Player search functionality</br>
🔔 Opponent notifications (challenge invites & game updates)</br>
📊 Competitive progression system</br>
📈 Ranking System</br>
Real-time matchmaking lobby system

**BeeFive uses an ELO-based system where:**</br>
- Winning increases your rating</br>
- Losing decreases your rating </br>
**You are placed on:**</br>
- Global leaderboard 🌍</br>
-Institutional leaderboard 🏫

**Platforms**</br>
- Web version built with Next.js</br>
- Mobile version built with Flutter (Dart)</br>
- Both platforms share the same competitive ecosystem and player data.

**Project Purpose** </br>
- BeeFive was built to create a competitive, skill-based multiplayer environment where players can:
- Compete globally
- Track progress over time
- Engage in structured ranking systems
- Challenge real opponents in real-time
