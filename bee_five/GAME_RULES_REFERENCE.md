# Adventure Game Rules Reference

This document lists which games have which special rules applied. All games are now unlocked for testing.

## Rule Categories

### 1. Mud Zones 🌊
**Games:** Multiples of 200
- Examples: 200, 400, 600, 800, 1000, 1200, 1400, 1600, 1800, 2000
- Effect: Mud zones are invisible areas that block moves during blind play

### 2. Blind Play 👁️
**Games:**
- Games ending in 42 pattern: 42, 92, 142, 192, 242, 292, 342, 392, 442, 492, 542, 592, 642, 692, 742, 792, 842, 892, 942, 992, 1042, 1092, 1142, 1192, 1242, 1292, 1342, 1392, 1442, 1492, 1542, 1592, 1642, 1692, 1742, 1792, 1842, 1892, 1942, 1992
- Multiples of 50, Match 2: 50 (match 2), 100 (match 2), 150 (match 2), 200 (match 2), etc.
- Effect: Cannot see mud zones, making them dangerous

### 3. Progressive Blocks 📈
**Games:** Ending in 3, from game 51+
- Examples: 53, 63, 73, 83, 93, 103, 113, 123, 133, 143, 153, 163, 173, 183, 193, 203, 213, etc.
- Effect: Blocks are added progressively every N moves based on game number

### 4. Disappearing Blocks 💨
**Games:** Ending in 4, from game 400+
- Examples: 404, 414, 424, 434, 444, 454, 464, 474, 484, 494, 504, 514, etc.
- Effect: Every 3 human moves, 2 blocked cells are removed

### 5. Shifting Blocks 🔄

- Ending in 7, from game 250+: 257, 267, 277, 287, 297, 307, 317, 327, 337, 347, 357, 367, 377, 387, 397, 407, etc.

- Ending in 8, from game 600+: 608, 618, 628, 638, 648, 658, 668, 678, 688, 698, 708, 718, etc.
  - Effect: Blocks shift every 5 moves

## Notes

- **Match Games**: Some rules only apply in specific matches (match 1, 2, 3, 4, or 5) of a best-of-3 or best-of-5 series
- **Multiple Rules**: Many games have multiple rules active simultaneously
- **Time Limits**: Decrease as game number increases (12s → 2s)
- **Starting Player**: Alternates between player and AI based on game number
